## ytsleep

This is an implementation of ytsleep in meteor
(not according to spec)

View this demo at http://ytsleep.beploy.se

### Install

Install Meteor
	$ curl https://install.meteor.com | /bin/sh

Clone and start
	$ git clone https://github.com/dumfan/ytsleep
	$ cd ytsleep
	$ meteor
